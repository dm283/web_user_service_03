INITIAL DEPLOY
установить по:
- Git (https://github.com/git-for-windows/git/releases/download/v2.48.1.windows.1/Git-2.48.1-64-bit.exe)
- Python [версия 3.10.8]  -  https://www.python.org/ftp/python/3.10.8/python-3.10.8-amd64.exe
  (при установке отмечается флажок "Add Path")
- NodeJS (https://nodejs.org/dist/v22.14.0/node-v22.14.0-x64.msi)
- Visual Studio Code (https://code.visualstudio.com/docs/?dv=win64user)
- postgresql (https://www.postgresql.org/download/,  https://www.pgadmin.org/download/)

для доступа к сервису из внешнего интернета нужно открыть порты 8000 и 5173

…

alembic init alembic
alembic/env.py:
	from app import models
	target_metadata = models.Base.metadata
	SQLALCHEMY_DATABASE_URL = f'postgresql://postgres:pwd@localhost/dbname'
	config.set_main_option('sqlalchemy.url', SQLALCHEMY_DATABASE_URL)
alembic revision --autogenerate -m "init (or any next)"
alembic upgrade head
